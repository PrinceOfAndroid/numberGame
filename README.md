# 数独小游戏（GitHub Pages）

这是一个纯前端静态项目，可直接部署到 GitHub Pages。

## 本地预览

直接打开 `index.html` 即可运行。

## 发布步骤

1. 在 GitHub 新建仓库（例如：`numberGame`）。
2. 把本目录代码推送到仓库 `main` 分支。
3. 打开仓库 `Settings` -> `Pages`，将 `Source` 设为 `GitHub Actions`。
4. 等待 Actions 工作流 `Deploy static site to GitHub Pages` 运行完成。
5. 访问：

`https://<你的GitHub用户名>.github.io/<仓库名>/`

示例：`https://octocat.github.io/numberGame/`
